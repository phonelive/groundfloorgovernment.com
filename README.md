# Ground Floor Government

Landing page for the Ground Floor Government podcast.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run generate
```

Static files are generated to `.output/public`.

## Deployment

- Push to `main` → deploys to dev.groundfloorgovernment.com
- Push a tag `v*` → deploys to groundfloorgovernment.com
